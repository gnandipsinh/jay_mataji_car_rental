# 🚗 Jay Mataji Car Rental Website

## 🌟 Overview

Jay Mataji Car Rental is a **modern, responsive, and premium car rental website** designed to provide users with an easy and smooth car booking experience.
The website focuses on **luxury UI design, user-friendly navigation, and responsive layout**.

---

## 🎯 Project Purpose

This project is created to:

* Provide online car rental services
* Showcase available vehicles (Fleet)
* Allow users to contact and book cars easily
* Build a strong digital presence for rental business

---

## ✨ Features

### 🔥 UI / UX

* Premium dark theme design
* Modern layout with clean spacing
* Smooth user experience

### 🚗 Fleet Section

* Car categories filter:

  * SUV
  * Sedan
  * Hatchback
* Detailed car cards:

  * Price per day
  * Fuel type
  * Seating capacity
  * Transmission type
* “Book Now” button

### 📞 Contact Section

* Contact form (Name, Email, Mobile, Message)
* Business details:

  * Phone number
  * Email address
  * Location
  * Working hours

### 📱 Responsive Design

* Mobile friendly layout
* Tablet optimized
* Desktop optimized

### 🧩 Sections Included

* Hero Section
* About Section
* Fleet Section
* Contact Section
* Footer Section
